## What This Project Is

**Warzone Lite:** A JavaScript RTS game for Axon BBS with:
- Multi-level terrain (0-8 elevations)
- Multi-altitude air combat (3 layers)
- Federated multiplayer via BBS events
- 15-20MB total size
